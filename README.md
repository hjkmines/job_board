# Job board for junior/entry-level tech jobs
## Setup

1. Clone the repo

```sh
git clone https://github.com/hjkmines/job_board.git
```

2. Install client node modules
From root level:
```sh
cd client
npm install
```

3. Repeat for server node modules

From root level:
```sh
cd server
npm install
```

4. Start client
From client directory:
```sh
npm run dev
```

5. Start dev server
From server directory:
```sh 
npm start
```
